Hier werden die Programmierungsskripte von der Masterarbeit dokumentiert. Wie in der Arbeit bereits erwähnt, handelt es sich bei
der Fragestullung in dieser Arbeit um drei hauptsächliche Scritte, nämlich die Vorverarbeitung von Daten, die extraktive Phase
und die abstraktive Phase. Die Skripte für diese drei Schritte werden separat in unterschidlichen Ordnern repräsentiert. 

Wenn Sie die Skripte anschauen bzw. benutzen möchten, bietet es hier zwei Möglichkeiten an: Sie können alle selbst mal laufen
laufen oder Sie können die Ergebnisse jedes Schritts bzw. die vortrainierten Gewichte (bei der abstraktiven Phase) direkt benuzten,
da das Training eher zeitaufwendig sein könnte. 

Hier werden alle Skripte angeboten und die Links für die vorverarbeitetn/vortrainierten Daten angebegen. Bei Anwendung dieser 
Skripte bitten ändern Sie die Pfaden für die Daten entsprechend, damit keiner Fehler auftreten wird. Die Likens leiten Sie zu 
meinem Google-Drive weiter, wo Sie alle nützlichen Daten finden können.

Nährere Informationen finden Sie in jedem Ordner noch.
