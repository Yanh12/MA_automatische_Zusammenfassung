Link for downloading the corpus: https://acl-arc.comp.nus.edu.sg/archives/acl-arc-160301-parscit/
1. use get_data.py file to get the data that are useful for the project 
2. use data_preprocessing.py file to preprocess the data, i.e. deleting unwanted features, fixing formats errors etc. and 
store them into in pickel-format (they can be in strings, sentences, or tokenized forms).
3. You can through the following links to access the final data from this step (basically the string-form, which are mainly
used in the following formats): 
 ----texts: https://drive.google.com/file/d/1bnZ4vC73FBA1prKSE1G7xF9KRhvBwxBn/view?usp=sharing
 ----abstracts: https://drive.google.com/file/d/1bnZ4vC73FBA1prKSE1G7xF9KRhvBwxBn/view?usp=sharing
